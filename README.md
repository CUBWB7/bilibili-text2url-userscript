# Bilibili使用增强-文本网址转链接（Text to URL）

一个面向 B 站的油猴脚本。

将视频简介、评论区、动态正文中的纯文本网址转换成可点击链接，并在新标签页打开；已经是链接的内容不会重复处理，`1.2万`、`3.14`、`v1.2.3` 这类文本也不会被误识别成网址。

当 B 站把评论区 URL 中的一段渲染成蓝色热词搜索链接时，脚本也会尽量恢复并生成完整的可点击网址。

为减少把文件名误判成网址的情况，脚本目前对“没有显式 URL 信号的裸二级域名”使用常见顶级域名白名单。例如 `soul.md` 默认不会转成链接，但 `https://soul.md`、`www.soul.md/docs` 这类写法仍会正常识别。

此脚本仅供个人学习与交流使用。

## 功能

- 支持视频简介中的纯文本网址
- 支持评论区中的纯文本网址
- 支持动态正文中的纯文本网址
- 支持页面动态加载后的新内容
- 自动跳过已有链接
- 对裸域名使用常见顶级域名白名单，减少文件名误判
- 支持恢复被 B 站热词搜索链接打断的完整 `http(s)` URL

## 安装

1. 安装 Violentmonkey 或 Tampermonkey
2. 打开 `dist/bilibili-text2url.user.js`
3. 新建脚本并粘贴内容，或直接导入
4. 保存后访问 B 站页面进行测试

## 开发

安装依赖：

```bash
corepack pnpm install
```

运行测试：

```bash
corepack pnpm test
```

构建脚本：

```bash
corepack pnpm build
```

构建完成后，可安装文件位于 `dist/bilibili-text2url.user.js`。
